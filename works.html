MOO DAY

JAVA BASED GAME PROJECT

This was my very first project ! Moo Day is a lively farming game built with Java and JavaFX, under the OOP concept. In this game, players can grow various crops, raise animals, and even sell their produce for profit. I took the lead on planning and developing from start to finish. It was an amazing experience watching Moo Day grow from just an idea to a fully playable game!

Box & Bone

Real-Time Web Application Game

Box & Bone is a fun game similar to Pac-Man, where players race to collect as many coins as possible within a set time limit! The player with the highest score will have their name displayed on the scoreboard. This game features real-time score updates, using MongoDB to store each player's scores. I have taken on the roles of task manager and full-stack developer for this exciting project!

Water Sensor

Water Level Measurement Project

This project is about a water level sensor used in a dam. I designed the entire website's UI/UX. To make it feel more realistic, I added animated water waves, and the container can be scaled to match real-life sizes. The water level changes based on sensor data, and there are alerts to warn when the level is too high. It was a fun experience to create these interactive features!
